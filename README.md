Use
===========

To run it you have to create a database in postgres with the name of

```
pokemon
```

then do 

```
npm install
npm start
```

you have to do it in the api and client directories.

The back loads on port 3001
```
localhost:3001
```

The front loads on the 3000

```
localhost:3000
```
